2. 서버 ot

서버의 종류에는 web server와 game server가 있다. web server는 테이크아웃 포장 전문 식당처럼 손님이 음식을 받아서 떠나면, 그 이후론 연락이 끊긴다. game server는 서빙 직원이 와서 손님에게 말을 걸 수도 있고 손님이 추가 주문을 하기도 하는 실시간 interaction이 된다.

game servers는 게임 장르에 따라서 요구사항이 달라서 최적의 프레임워크라는 것이 존재하기 애매하다.

mmorpg 게임 서버 제작은 식당 운영에 비유가 가능하다.
 - 멀티쓰레드	: 직원들을 고용하고 운영하는 것.
 - 네트워크		: 직원들이 손님과 어떤 방식으로 대화를 할지.
 - DB			: 결제 및 장부 처리를 어떤 방식으로 할지
 - 게임 로직 	: 손님들의 주문을 신속하고 맛있게 요리하는 것.
 - 서버 구조	: 위에서 정한 정책에 맞는 식당의 인테리어에 대해 고민하는 것.
